# 1. Project Overview

## 1.1 Project Name

**DaDamDa (다담다)**

Meaning

> 모든 가치를 담다.
>
> 전통시장의 음식과 사람, 그리고 지역을 하나로 연결하는 플랫폼

---

# 1.2 Vision

다담다는 단순한 할인 플랫폼이 아니다.

우리는

"전통시장의 디지털 운영 플랫폼(Market Operating System)"

을 구축하는 것을 목표로 한다.

초기 MVP는

'마감 할인 예약 플랫폼'

으로 시작하지만

최종적으로는

- 재고관리
- 할인관리
- 예약관리
- 단골관리
- AI 수요예측
- AI 가격추천
- ESG 데이터 분석

을 하나의 플랫폼에서 제공하는 것이 목표이다.

---

# 1.3 Mission

전통시장에서 발생하는

"판매 가능한 재고의 폐기"

를 줄이고

소비자에게

합리적인 소비 경험을 제공한다.

---

# 1.4 Tag Line

> "오늘의 남은 가치,
> 다담다."

---

# 2. Problem Statement

## 현재 전통시장의 문제

매일 영업 종료 전

수많은 음식들이

폐기된다.

예시

- 반찬
- 떡
- 생선
- 정육
- 과일
- 분식

이는

품질이 나빠서가 아니라

마감시간 전에 판매되지 못했기 때문이다.

---

### 상인의 문제

- 재고 처리 어려움
- 폐기 비용 발생
- 할인 홍보 수단 부족
- 디지털 판매 경험 부족

---

### 소비자의 문제

- 할인 정보를 실시간으로 알 수 없음
- 퇴근 후 장보기 어려움
- 시장마다 직접 방문해야 확인 가능

---

### 사회적 문제

- 음식물 폐기 증가
- 탄소배출 증가
- 전통시장 경쟁력 감소

---

# 핵심 문제 정의

우리가 해결하려는 문제는

> "음식이 남는 것이 아니라,
> 정보가 연결되지 않는 것이다."

---
